Rules for Quiz

1. Each question carries one mark.

2. Each correct answer will be awarded 2 Mark.

3. One mark will be deducted for each wrong answer given.

4. No mark will be deducted for question left unanswered.

5.The participants shall not be allowed to use mobile or other electronic instruments

6. Answering time is only 30 seconds per question.
